### To start the project 
1. Create vite project
```js
   npm create vite@latest
  },
```
2. Clone the project
3. Install the packages listed above
4. Under "src/constants/" Configure the URL to the server domain/api
5. Start the Project
```js
   npm run dev
  },
```
